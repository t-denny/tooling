### Installation

Just run start.sh.

```sh
$ ./start.sh
```

### Verify installation
Verify the deployment by navigating to your server address in your preferred browser.

```sh
127.0.0.1:5001/vm
```

### How to use

Just use.

| Endpoint | Method |
| ------ | ------ |
| 127.0.0.1:5001/vm | GET / POST |
| 127.0.0.1:5001/vm/<id> | GET / PUT / DELETE |
| 127.0.0.1:5001/vm/type/<instance_type> | GET |
